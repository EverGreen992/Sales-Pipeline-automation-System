# Sales Pipeline & Task Automation System

## Project Overview
This project demonstrates the design and implementation of a structured sales workflow using HubSpot CRM, ClickUp, and Make.com.
The system ensures automated follow-up tracking, task creation, and improved deal visibility.

## Tools Used
- HubSpot CRM
- ClickUp
- Make.com

## Key Features
- Custom sales pipeline configuration
- Automated follow-up task creation
- Cross-platform workflow automation
- Structured sales SOP

## Business Value
- Prevents missed follow-ups
- Improves response time
- Standardizes sales operations
- Enhances deal tracking visibility
